## CMD Aliases
#### Configure Aliases

```
    $ doskey /macrofile=[path to cmd-macroses]         # Import Aliases
    $ doskey /macros > [path to cmd-macroses]          # Export Aliases
```
#### GIT
``` 
   $ gcb [base branch name] [new branch name] - checkout new branch from lates in base branch 
```
